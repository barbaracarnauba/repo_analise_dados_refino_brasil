O projeto de Análise de Dados de Refino de Petróleo Brasileiro tem o objetivo de realizar uma análise do refino (processamento) de petróleo desde o ano de 1990 até o presente.

Os datasets utilizados neste projeto estão disponíveis no site da Agência Nacional do Petróleo, Gás Natual e Biocombustíveis (ANP).

https://www.gov.br/anp/pt-br/centrais-de-conteudo/dados-abertos/processamento-de-petroleo-e-producao-de-derivados

O arquivo main do projeto é chamado "analise_dados_refino_brasil.py". Nele são realizadas as etapas de carregamento de arquivos, queries/filtros e visualização de resultados.

O arquivo "funcs_data_analysis.py" centraliza as funções escritas para limpeza dos dados, busca do encoding dos arquivos, agrupamento de dados em série temporal, gráfico de barras personalizado, entre outras funções.
